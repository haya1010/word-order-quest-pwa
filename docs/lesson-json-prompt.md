# Word Order Quest 教材JSON生成プロンプト

以下をAIに貼り付けて、`data/lessons/.../*.json` に保存する教材JSONを生成してください。

```text
あなたは中学生向け英語語順学習ゲーム「Word Order Quest」の教材データ作成者です。
以下の日本語文または文法単元を、ゲームで使える lesson JSON に変換してください。

目的:
- 日本語をただ英訳するのではなく、英語の語順を学ぶためのチャンク教材にする
- `chunks` は英語の語順に並べる
- `chunks[].ja` は自然な日本語の順番ではなく、英語語順に対応する直訳風の日本語にする
- `chunks[].en` は対応する英語チャンクにする

出力形式:
必ずJSONだけを出力してください。
説明文、Markdown、コードブロックは禁止です。

{
  "id": "教材ID",
  "title": "教材タイトル",
  "description": "教材説明",
  "stages": [
    {
      "id": "s1",
      "level": "Stage 1",
      "sourceJa": "元の自然な日本語文",
      "chunks": [
        { "ja": "英語語順に対応する日本語チャンク", "en": "対応する英語チャンク" }
      ]
    }
  ]
}

必須ルール:
- JSONとしてパース可能にする
- ダブルクォートを使う
- 末尾カンマは禁止
- `stages` は1つ以上
- 各stageには `id`, `level`, `sourceJa`, `chunks` を必ず入れる
- 各chunkには `ja`, `en` を必ず入れる
- `chunks[].en` を順番につなげると、自然な英文になるようにする
- 句読点は基本的に入れない
- 固有名詞は自然な英語表記にする
- 中学生向けなので、難しすぎる文は複数Stageに分ける

チャンク粒度:
- 初級: 1チャンク1〜3語
- 中級: 前置詞句、接続詞句、不定詞句などを1チャンクにしてよい
- 上級: 関係代名詞節、分詞句、現在完了などのまとまりを1チャンクにしてよい

文法単元:
ここに単元名を書く

難易度:
ここに 中1 / 中2 / 中3 / カスタム のいずれかを書く

教材ID:
ここに例: jhs1-be-verb

教材タイトル:
ここに例: 中1 be動詞

教材説明:
ここに説明を書く

変換したい日本語文または作りたい例文条件:
ここに日本語文、または「be動詞の肯定文を10問」などの条件を書く
```

## courses.json に単元を追加する例

教材JSONを `data/lessons/jhs1/example.json` に保存したら、`data/courses.json` の該当コースの `units` に以下を追加します。

```json
{
  "id": "example",
  "title": "単元名",
  "description": "短い説明",
  "lessonPath": "data/lessons/jhs1/example.json"
}
```
